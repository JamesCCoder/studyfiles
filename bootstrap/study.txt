7. button /  button-group    (project_6)
 
 *** button type: button submit reset  (submit and reset are using in forms to submit or reset data that from form)

class="btn"                     默认的 btn 无背景色，无边框
class="btn btn-primary"         生成有背景色的 btn， 无边框,文字颜色默认为白色
class="btn btn-outline-primary" 生成无背景色 btn， 有边框，边框色和文字色一致

class="btn btn-sm"              btn-sm  btn-lg   改变 btn 大小

disabled                        禁用 btn
active                          激活状态  btn

插入 <span> 实现效果:
   
        <button class="btn btn-primary">
            <span class="spinner-border spinner-border-sm"></span> Loading...
        </button>

     spinner-border      spinner-grow
     spinner-border-sm   spinner-grow-lg


<div clsss="btn-group">
    <button class="btn btn-primary">This is a button</button>
    <button class="btn btn-success">This is a button</button>
    <button class="btn btn-warning">This is a button</button>
</div>

其他： d-flex               两端对齐
       btn-group-vertical   垂直排列
