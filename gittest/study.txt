10. merge branch

    git merge newOne                          将newOne merge到当前branch

                                              三种不同结果：
                                              1. fast forward (changes in one branch cover other changes in another branch)
                                              2. need commit but no conflict (create new file)
                                              3. has conflict (edit the conflict part, keep what we need and commit it)
   

11. stash (stash 可以多次使用，产生多个缓存)

    git stash                                 取消上次commit之后的一切变化并保留在stash cache中
    git stash pop                             将stash中内容放回原位置并释放缓存
    git stash apply                           将stash中内容放回原位置，不释放缓存。

                                              pop 和 apply 可能会产生三种后果，
                                              处理方法和merge相同
                                                  
    git stash list                            列举所有的stash
    git stash drop                            清除某个/最近一个stash
    git stash clear                           清楚全部stash
